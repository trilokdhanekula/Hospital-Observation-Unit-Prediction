# Hospital-Observation-Unit-Prediction

## Overview
This project focuses on predicting inpatient transfers from hospital observation units using machine learning techniques.

## Objective
To identify patients at high risk of requiring inpatient admission and improve hospital resource allocation and patient flow.

## Tools & Technologies
- R
- Logistic Regression
- Random Forest
- R Markdown
- Data Visualization
- Excel

## Dataset
The dataset contains patient-level clinical and operational variables including:
- Age
- Blood Pressure
- Pulse
- Temperature
- Length of Stay
- DRG codes

## Process
- Data cleaning and preprocessing
- Handling missing values
- Exploratory Data Analysis (EDA)
- Logistic Regression modeling
- Random Forest modeling
- Model evaluation using confusion matrix and accuracy metrics

## Results
### Logistic Regression
- Accuracy: 77.27%

### Random Forest
- Accuracy: 75.45%

## Business Impact
This project demonstrates how predictive analytics can help hospitals:
- Improve patient flow
- Optimize observation unit utilization
- Reduce emergency department congestion
- Support proactive patient management
